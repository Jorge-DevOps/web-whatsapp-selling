🚀 EMPEZAR EN 3 PASOS
════════════════════════════════════════════════════════════════════════════

PASO 1: Instalar (2 minutos)
────────────────────────────
  $ cd whatsapp-selling-bot
  $ npm install

PASO 2: Ejecutar (1 minuto)
─────────────────────────────
  $ npm start
  ✅ Se mostrará un código QR en la terminal

PASO 3: Autenticar (2 minutos)
──────────────────────────────
  1. Abre WhatsApp en tu celular
  2. Ve a Configuración → Dispositivos conectados → Conectar dispositivo
  3. Escanea el código QR con la cámara de tu celular
  4. Espera a que muestre ✅ en la terminal

⏱️ TOTAL: ~5 minutos para tener bot funcionando
